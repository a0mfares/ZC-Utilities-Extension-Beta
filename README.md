# ZC Toolbox

An Extension for Zewail City students to help with GPA calculation, coursework management, and academic planning.

## Features

- **GPA Calculator**: Easily calculate Cumulative GPA.
- **Coursework Calculator**: Track your grades and coursework progress.
- **Academic Planner**: Plan your courses for upcoming semesters.
- **Stop List Blocker**: Automatically handles the stop list grade blocking to improve user experience on the SIS.

## Installation

You can install this extension manually on Chrome, Edge, Brave, or Firefox.

### Video Tutorial
Watch this video to learn how to install the extension on Google Chrome and Firefox:
[How to Import ZC Toolbox Extension](https://www.youtube.com/watch?v=dhaGRJvJAII)

### Manual Installation (Chrome / Edge / Brave)
1. Download or clone this repository to your computer.
2. Open your browser and go to the Extensions management page:
   - Chrome: `chrome://extensions`
   - Edge: `edge://extensions`
   - Brave: `brave://extensions`
3. Enable **Developer mode** (usually a toggle in the top right corner).
4. Click **Load unpacked**.
5. Select the folder containing the extension files (the folder with `manifest.json`).

### Manual Installation (Firefox)
1. Download or clone this repository.
2. Open Firefox and go to `about:debugging`.
3. Click on **This Firefox** in the sidebar.
4. Click **Load Temporary Add-on...**.
5. Select the `manifest.json` file from the extension folder.
